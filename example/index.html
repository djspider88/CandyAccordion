  • What is CandyAccordion?
    CandyAccordion is a small, lightweight and easy to use accordion that takes advantage of the jQuery library.
  • Who Made It?
    CandyAccordion was made by @steve228uk, a web developer from Leicester, UK.
  • Can I Use CandyAccordion?
    Absolutely! CandyAccordion is under a dual MIT/GPL license (the same as jQuery) meaning anyone can use, adapt and contribute to it. Of course I wouldn't mind a link back.
  • What Version of jQuery Does it Require?
    CandyAccordion should work with jQuery 1.3+ but if you're not using the latest version I'd strongly recommend upgrading as there are some nice new functions.
  • Anything Else?
    No, that's all for now. Take a look at some of my other plugins if you're looking for a simple image gallery or tab plugin.
